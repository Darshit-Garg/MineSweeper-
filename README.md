# Minesweeper (JavaFX)

This repository contains a **Minesweeper game built using JavaFX**, with **Maven** as the build architecture.  
The project recreates the classic puzzle game where players uncover tiles and avoid hidden mines.

## Features
- 🎮 Interactive Minesweeper gameplay
- ⏱ Built-in timer to track performance
- 🖥️ JavaFX-based graphical user interface
- ⚙️ Maven for dependency management and build automation
- 🧩 Customizable grid and difficulty levels

## Requirements
- Java 8 or higher
- Maven 3.6+

## Installation & Running
1. Clone the repository:
   ```bash
   git clone https://github.com/Darshit-Garg/MineSweeper-.git

2. Navigate to the project directory:
   ```bash
   cd MineSweeper-

3. Build the project with maven:
   ```bash
   mvn clean install

4. Run the application:
   ```bash
   mvn javafx:run

## Project Structure
- `src/main/java` → Core game logic and UI  
- `pom.xml` → Maven configuration  
- `.gitignore` → Project ignores
